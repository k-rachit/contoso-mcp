# A2A Agent Layer\n\nExposes the Contoso CRM data as a Google A2A-compatible agent over HTTP.\n\n## Why Both MCP + A2A?\n\n| Protocol | Purpose | Client |\n|----------|---------|--------|\n| MCP | Human-to-agent (copilot) | Perplexity Mac app |\n| A2A | Agent-to-agent (multi-agent) | Any A2A client, LangGraph, ADK |\n\nSame data, two interfaces.\n\n## Run\n\n    cd a2a-demo\n    node a2a-server.js\n\n## Test\n\n    curl http://localhost:3001/.well-known/agent.json\n\n    curl -X POST http://localhost:3001/tasks/send -H "Content-Type: application/json" -d '{"id":"1","message":{"role":"user","parts":[{"type":"text","text":"Brief me on Northwind"}]}}'\n
